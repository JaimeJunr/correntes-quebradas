# Correntes Quebradas — início de jogo para Kenshi

Um novo **início de jogo (game start)** para [Kenshi](https://store.steampowered.com/app/233860/Kenshi/), focado em vingança e na luta contra a escravidão, fiel à lore do jogo.

> Você era um guerreiro Shek, com esposa e filha. Então vieram os escravagistas das Cidades Unidas: quem você amava tombou defendendo a casa, e sua filha foi acorrentada e arrastada para os mercados de carne do Império. Hoje você acorda nas valas de Squin, sem uma moeda, com uma única certeza fria: arrancar a escravidão deste mundo pela raiz.

## Características

- Início único **"Correntes Quebradas"** — dificuldade **Difícil**, mas jogável.
- Personagem **Shek** (ex-guerreiro decaído), stats moderados-baixos: força 10, combate corpo-a-corpo 20, desarmado 10, à distância 5, furtividade 5.
- Começa em **Squin** (Reino Shek), refúgio seguro perto dos territórios escravagistas.
- Sem dinheiro, equipamento básico.
- **Reputação inicial balanceada** (difícil, não impossível):

| Facção | Relação |
|---|---|
| Mercadores de Escravos | -50 |
| Caçadores de Escravos | -50 |
| Caçadores de Homens | -40 |
| Nação Sagrada | -30 |
| Cidades Unidas | -20 |
| Saqueadores | -20 |
| Anti-Escravagistas (Tinfist) | +40 |

## Instalação

**Via arquivo (manual):**
1. Copie a pasta para `Kenshi/mods/CorrentesQuebradas/` (o `.mod` deve ficar dentro de uma pasta com o mesmo nome).
2. Ative **CorrentesQuebradas** no launcher do Kenshi.
3. Novo Jogo → escolha **"Correntes Quebradas"**.

Compatível com saves novos. Sem dependências além do jogo base.

## Mods opcionais recomendados (não obrigatórios)

- **Minor Faction Overhaul – The Slave Trade** — mais escravagistas no mundo.
